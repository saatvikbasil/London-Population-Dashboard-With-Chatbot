API Access

For developers and researchers, our data is also available through a REST API. You can programmatically access the same population data using the following endpoints:

Endpoint Description Parameters
/api/population Get population data with optional filters location_id, demographic_id, gender_id, year_id, metric_id, page, per_page
/api/population/summary Get summary statistics for population data location_id, demographic_id, gender_id, metric_id
/api/population/trends Get population trends over time location_id, demographic_id, gender_id, metric_id
/api/locations Get all available locations None
/api/years Get all available years None
Example API Request
GET /api/population?location_id=1&year_id=10
Content-Type: application/json

Response:

  "data": [
