Upload Guidelines

File Format Requirements

  • Files must be in CSV or Excel (.xlsx) format
  • Maximum file size is 10MB
  • The first row must contain column headers
  • Required columns depend on the selected data type

Column Requirements

Data Type Required Columns
Population Data location_id, demographic_id, gender_id, year_id, metric_id, value
Location Data location_id, location_name, region (optional)
